Microsoft's agent memory system uses 98% fewer tokens on long tasks

Today's agents restart from zero each session. Microsoft Research's Memora, published at ICML 2026 with code public on GitHub, splits memory into rich stored content and lightweight retrieval tags, letting agents navigate months of history using up to 98% fewer context tokens than full-history replay. Does a memory layer this cheap flip agent infrastructure to build-versus-buy, and who decides?

A Quantpedia experiment sent an autonomous research agent to replicate nine published US-equity anomalies on survivorship-free data. Measurement is the gap. Today's signal sits in Currents because it rests on a single blog post, not peer review. The direction it points at is real: clinical AI is being judged on the wrong question entirely.

Healthcare AI benchmarks measure the wrong thing, a 61-point gap shows

The 61-point drop in healthcare model accuracy between evaluation and deployment isn't a design flaw; it's unstated assumptions. Evaluations ask if the model answers correctly; deployments ask if patients act on it. A CMU blog post proposes BenchmarkCards to map that gap. Standard practice makes it a priceable cost; unmapped, it keeps reading as model failure, not diagnosis failure.
