The Pulse · Story of the day

ChatGPT now blocks the final step in prompt-injection data theft

Prompt injection attacks follow a three-step pattern: reach private data, manipulate the model, exfiltrate the result. OpenAI's Lockdown Mode cuts the third leg by blocking outbound network requests deterministically, without relying on AI judgement that can itself be subverted. It doesn't stop injections entering; it stops data leaving. Default ChatGPT settings offer no such guarantee.

Architecture is doing the heavy lifting. Across claims processing, code generation, and correctness verification, the performance gains being reported trace back to structural choices beneath the model, not the model itself. That shifts the vendor conversation.

Structured agent logic cuts token use 30-fold, IBM tests show

IBM Research argues that layering structured logic beneath a language model slashes costs and errors. Their legacy-code agent consumed 30 times fewer tokens than a frontier-only baseline; a test-generation tool hit 20-45% better code coverage. All benchmarks are vendor-supplied. Ask for independent replication before committing your agent architecture.

Formal proof may be the missing ingredient for AI that compounds

Axiom scored 187/189 (99%) on a code-correctness benchmark where OpenAI's leading reasoning model scored 4.9%. The argument: formal proof verification gives reinforcement learning a cleaner signal than statistical methods, letting gains compound rather than plateau. Vendor-reported, no independent replication yet. If you're setting AI research priorities for the next six months, this framing deserves a serious look.
