Agents face a security threat that cybersecurity tools cannot catch

Zico Kolter (OpenAI board) and Gray Swan CEO Matt Fredrikson argue agents introduce a genuinely new exploit class: prompt injection, where untrusted data hijacks an agent's actions. Gray Swan's automated red-teaming model already outperforms humans at breaking AI systems. Single-source podcast, no independent benchmarks. Does your red-teaming programme treat agent security as distinct from traditional software security?

Trading AI backtests collapse when market impact costs are finally counted

Most reinforcement learning trading agents backtest brilliantly because the simulation assumes a flat ten-basis-point fee regardless of trade size or speed. A new paper rebuilds the cost model with realistic market impact, and the apparent top performers fall apart. Single-source, no code released. Before trusting any backtest ranking, ask whether execution costs scale with position size.

Quant tooling is getting sharper. Three independent research threads are quietly closing gaps that kept ML out of live derivatives work. Each sits here because none has code, replication, or adoption yet.

Options models can now train on price and volatility targets simultaneously

Early signal from a quant ML paper sweep: PIVOT embeds a differentiable volatility-inversion step into model training, letting a network optimise against price and volatility targets jointly. Single paper, no public code mentioned, thin sourcing. If the technique replicates, it tightens options model calibration; if it doesn't, the benchmark gap between price-space and vol-space objectives stays open.

Giving a language model thinking time helps it recall facts it otherwise misses

Google Research found that reasoning traces unlock facts a model holds but cannot surface directly, via two mechanisms: extra tokens act as a compute buffer, and generating related facts primes correct recall. One hallucinated intermediate fact measurably reduces final accuracy. No code released; if this replicates, it changes how you prompt for factual retrieval. If it doesn't, the audit methodology is still the useful artefact.

A derivatives pricing technique now covers options it previously excluded

Differential machine learning prices derivatives fast by training on sensitivities alongside prices, but broke on discontinuous payoffs like barriers and digitals. Glasserman and Karmarkar extend it to cover those cases. No code released; paper is paywalled. If the approach replicates, exotic-desk pricing pipelines gain a faster approximation; if not, the barrier-option gap stays open.
