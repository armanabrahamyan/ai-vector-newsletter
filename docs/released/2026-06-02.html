Computer-use agents can now run locally, fast enough for production

Holo3.1 ships quantised weights that run a computer-use agent entirely on local hardware, hitting 79.3% on AndroidWorld's mobile benchmark, up from 67% in the prior version. AI agents leak user intent to outside services before they act

Agents fire speculative tool calls to cut latency, leaking inferred user intent to external services before committing. Access controls cannot unsend what an observer already holds; only suppressing the call before dispatch works. Single arXiv paper, prototype runtime, no public code. Replace looping search calls with model-written code to cut tokens 85%

Perplexity's 'Search as Code' pattern has the agent write Python to compose each retrieval pipeline on the fly, rather than firing repeated fixed search calls. On a 200-CVE task it cut token use 85% at full accuracy. Benchmarks are vendor-supplied; the SDK is private. Try the pattern via the public Hermes Agent walkthrough before your next agentic retrieval build.

Telling your forecasting model a regime shift is coming cuts error 25%

Early signal from University of Michigan research: transformers can detect mid-sequence regime shifts in-context, and prompting them with an explicit change-signal cuts forecasting error by 25%. Single study, no public code yet. If you're evaluating AI forecasting tools for trading or macro signals, ask vendors whether their models handle non-stationary data, or this gap is yours to own.

Meta's Instagram support chatbot let attackers seize accounts by asking it to swap the linked email address, bypassing identity checks entirely.
