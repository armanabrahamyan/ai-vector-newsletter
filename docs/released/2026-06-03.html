Language models are unreliable safety judges for financial advice

Language models used as automated safety judges disagree with each other and themselves on financial-advice harm, missing subtle violations while flagging obvious violence reliably. Inconsistency also shifts by writing style and language. Single paper, no accompanying code. Measure inter-judge disagreement before treating any one model's safety verdict as authoritative in a regulated eval pipeline.

Multi-agent AI systems reach consensus by forgetting the facts

When multiple language models deliberate, they tend to agree more while knowing less. A new diagnostic framework, DelibTrace, tracked issue-critical facts across discussion rounds and found up to 72% were lost before consensus formed. No code is confirmed available yet. If you're designing multi-agent systems for research, analysis, or decisions, consensus is not a reliability signal.

Methodology before metric. Today's signal is a single public dataset, no code attached. The practitioner takeaway is structural: how you score AI outputs matters more than which model you score.

New legal benchmark finds pairwise scoring beats rubrics for AI evaluation

JudgmentBench pairs 1,539 rubric scores with 1,530 pairwise preference judgements from practising attorneys across 30 legal tasks. Pairwise scoring recovered the intended quality ranking with a Spearman correlation of 0.908; rubrics managed just 0.150. Dataset is public; no code repo yet. Switch your language-model judge pipeline to pairwise comparisons before your next evaluation design.
