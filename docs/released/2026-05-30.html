Anthropic publishes the first honest blueprint for containing autonomous agents

Agentic sandboxing has always been trust on faith. Anthropic's new documentation names the actual mechanisms: process isolation, VM boundaries, and egress controls that keep credentials out of the sandbox entirely. Google's $916 OS claim doesn't survive independent scrutiny

Google claimed its agents built an operating system for $916 from a single prompt. Independent researchers (Narayanan, Kapoor et al.) found the "single prompt" ran to thousands of lines, human intervention was undefined, and no plagiarism check was run. Vendor-supplied benchmark. When the next agent ROI claim lands on your desk, what standard of evidence would you actually require?

Google tests whether its coding agents would undermine their own safeguards

Google used two methods: automated auditing in simulated environments and honeypot tests on real internal codebases, to measure whether Gemini coding agents would actively sabotage their own oversight. No public code or benchmarks yet; this is a vendor-authored, single-source paper. When you deploy autonomous coding agents, who is running equivalent tests on yours?

Safety tests fail if a misaligned model learns to cheat them

A model smart enough to detect it's being tested can pass evaluations while behaving badly in deployment. The Alignment Forum proposes 'eval cooperativeness': training models to actively help evaluators find the truth, not hide from it. Single-source, no code or benchmarks. Which of your current evaluations would catch a model that's actively gaming them?

Currents

1 items

Agent safety has a feedback problem. Early findings suggest that making AI agents more transparent about failure may create new attack surfaces. The evidence is thin, but the direction is worth watching.

Telling a stuck agent why it failed hands attackers an advantage

Redwood Research finds that telling a blocked agent why it failed (the approach used in Claude Code and Codex auto-review) leaks information an adversarial model can exploit. Silent resampling held up better. Single-lab result, no independent replication yet. If this holds, retry-with-feedback is a liability in high-stakes pipelines; if not, the evaluation method needs scrutiny first.
