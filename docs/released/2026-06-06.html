Open benchmark tests voice agents across 213 realistic enterprise scenarios

ServiceNow's EVA-Bench 2.0 covers 213 voice-agent scenarios across airline, IT, and healthcare workflows, spanning 121 tools. Every scenario has exactly one correct resolution path, reducing evaluation noise. Datasets are open and downloadable today. Vendor-validated only, but a rigorous open eval for enterprise tool-use has been overdue; this is the most concrete one available.

The Big Picture

3 stories

Measurement is breaking down. Across today's stories, the tools used to count AI progress -- accuracy benchmarks, GDP statistics, threat models -- are each failing to capture what is actually changing. The gap is strategic, not technical.

Google's enterprise agent keeps searching until it finds a complete answer

Standard retrieval-augmented generation stops at the first search; Google's new multi-agent framework on its enterprise platform keeps iterating until a "Sufficient Context Agent" confirms the answer is complete. Vendor-reported accuracy lift is 34% on factuality datasets, but benchmarks are Google's own. If you're designing query pipelines across siloed data sources, raise this architecture pattern at your next design review.

A new worm carries its own AI brain onto machines it infects

Researchers have prototyped a worm that bundles a language model inside itself, running it on each compromised machine: payload and reasoning engine as one unit. Single source, no code or paper published. If you run AI workloads on internet-facing infrastructure, ask whether host-based model execution sits in your current threat model.

Hands-On

5 stories

Scaffold before you score. Today's pattern is infrastructure before evaluation: safe execution environments, persistent agent state, and trace-level grading are each moving from workaround to installable primitive. The benchmark gap is real, but so are the tools closing it.

Run LLM-generated Python safely in a WebAssembly sandbox you can pip-install today

Safe code execution for AI agents is a persistent gap. Simon Willison's alpha package micropython-wasm runs Python inside a WebAssembly interpreter, enforcing memory, CPU, and network limits with no host-system access. A live Datasette Agent plugin is already using it. Alpha-stage and self-described as 'vibe-coded', so clone it against a non-production agent before trusting it with user inputs.

Grading your agent on its final answer hides how it actually ran

Harness-Bench ran 5,194 agent trajectories across 106 tasks and found the scaffolding wrapped around a model moves performance more than the model itself does on hard tasks. Single-source summary, no public repo yet. Before your next agent evaluation, define what your trace must log at each layer; a green result without execution telemetry is a receipt, not a review.

One researcher's 2026 paper shortlist maps where language models are heading

Sebastian Raschka's Jan–May 2026 reading list organises notable language model papers into 10 categories, covering agent harnesses, diffusion language models, reasoning, and inference efficiency. Raschka flags it as personal and incomplete, not exhaustive. Before your next architecture planning session, use the agent systems and inference sections to pressure-test your paper backlog.

A second training pass cuts document-reading failure loops by 59%

Direct Preference Optimisation, typically used for chat alignment, works on objective structured tasks too. DharmaOCR applied it after supervised fine-tuning on document extraction, using the model's own degenerate outputs as rejection pairs. Degeneration fell 59.4% on average across every model family tested. Single-vendor benchmark. If repetition loops are fouling your document extraction pipeline, add a DPO stage after fine-tuning.

try huggingface.co →

Vercel's sandbox tool now keeps agent workspaces alive between sessions

The persistent-storage gap in disposable sandboxes was forcing agents to rebuild state from scratch on every run. Vercel Drives lets a sandbox mount a named drive at a configurable path; the drive survives when the sandbox stops. Private beta, single read-write mount per drive. Install @vercel/sandbox@beta and request access before wiring it into an agentic pipeline.
