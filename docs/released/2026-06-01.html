Europe's AI enforcement clock runs out in nine weeks

The EU AI Act's 2 August enforcement deadline is nine weeks out, and national implementation is still patchy. Poland just adopted a draft oversight bill; most member states haven't. The EU's AI Office confirms formal powers activate then. Single-source reporting from a Future of Life Institute newsletter, but the direction is clear even so.

Capability outruns control. AI systems are proving they can do more than anyone expected, yet the governance structures needed to bound their behaviour, and their data exposure, are still catching up.

Deep research agents leak private data through their own web searches

When an agent searches the web using private documents as context, the queries themselves become the leak. MosaicLeaks (1,001 tasks, arXiv preprint) found leakage in over a third of cases; a privacy-aware training approach cut that to under 10%. No code yet. If your agents mix client documents with external calls, query logs are a confidentiality exposure.

A blueprint for auditable AI agents in regulated security operations

Security operations analysts using AI agents can bind their organisation legally with a single tool call, yet no standard runtime enforces scope across retrieval, memory, and audit. This paper proposes a typed Security Context enforced at every component boundary, with append-only audit logs and tiered human approval gates. No code released; raise it before your next agentic architecture decision.

AI cracked a geometry problem that defeated mathematicians for 80 years

OpenAI's model disproved the Erdős unit distance conjecture, a discrete geometry problem open since 1946, by assembling existing ideas across mathematical subfields rather than inventing new ones. Fields Medal winner Tim Gowers called it 'a milestone.' The result has since been extended by human mathematicians. When does AI stop synthesising known mathematics and start generating new frameworks?

Verify before you deploy. Today's releases are genuinely capable, but every benchmark here comes from the team that built the tool. Independent validation is the missing step before any of this reaches production.

One open model now handles robotics reasoning and video generation together

Cosmos 3 (two sizes, weights on Hugging Face) unifies video generation, physical reasoning, and robot-policy output in a single architecture, replacing four separate prior models. Post-training scripts and synthetic datasets ship on GitHub. Benchmarks come from NVIDIA's own materials only. Clone the Nano variant against your simulation pipeline before committing to the larger Super model.

Every frontier model scores below 50% on this financial spreadsheet test

BlueFin (arXiv:2605.30907) is an open-source benchmark of 131 finance-specific spreadsheet tasks, validated by human annotators; its judge reaches near-expert agreement (Krippendorff's alpha of 0.826). Every frontier model tested scored below 50%. Before deploying a language model agent on financial analysis workflows, run it against BlueFin's harness to expose dynamic-correctness gaps first.

JetBrains open-sources a coding model that runs on 2.5B active parameters

Mellum 2 is JetBrains' open-weight coding and agentic model: 12B total parameters but only 2.5B active per token, keeping inference cheap on commodity GPUs. Weights, an instruct variant, and a reasoning variant are all Apache 2.0. Benchmarks come from the authors' own report, so run it against your current coding-agent eval before trusting the claims.
